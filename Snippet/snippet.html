The Anatomy of an HTML Tag

In this section I am going to breakdown a snippet of code into the various components that make it up and explain what each part does and how the different parts relate to each other.

TAGS

Tags are created by the use of less-than and greater-than angle brackets around an element.Tags occur in pairs of opening and closing tags. In the above we are going to identify the opening and closing tags

Opening Tags

An opening tag marks the beginning of an element. It starts with a less-than sign followed by an element’s name, and then ends with a greater-than sign.

Closing Tags

A closing tag marks the end of an element. It starts with a less-than sign followed by a forward slush, an element’s name, and then ends with a greater-than sign.
